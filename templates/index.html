AI Music Generation

Recurrent Neural Network(LSTM) based music generation through AI/ML


A web application service that integrates an AI/ML model which allows individuals and organizations to generate piano music based on a stacked LSTM based RNN model.





Generate New Music Here

The AI/ML model will generate a new series of piano music based on the model's learning on the sequential music data.

It will take about 10-20 sec for model to generate music. The file will get automatically downloaded on your system.

About The Model

The model is an instance of deep LSTM Recurrent Neural Networks.

It has the power to generate new notes starting from a random note to give a completely new song everytime you generate one.

About The Dataset

The dataset contains .mid or midi files to extract chords and notes from a particular file. I created the dataset for this model by doing web scraping.
